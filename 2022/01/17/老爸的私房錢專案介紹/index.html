老爸的私房錢專案介紹

為何會選擇這個專案？

老爸的私房錢專案包含最基本的 CRUD 和登入登出功能，資料庫方面有簡單的資料關聯，是適合做為全端開發練習的專案。

使用了什麼技術？

  • 以 JavaScript 最為開發語言
  • 以 Express 做為開發框架，採用符合 RESTful 的路由設計，並將 View 和 Model 各自獨立分開
  • 以 MongDB 作為 NoSQL 資料庫，採用 momgoose 作為 ODM 使用 Schema 、 populate 等功能
  • 以 handlebars 為樣板引擎，使用 handlebars 提供的 helper 做簡單的資料處理和邏輯判斷
  • 以 passport 製作使用者認證系統，使用 bcryptjs 確保密碼儲存的安全性
  • 以 connect-flash 實作登入、登出、註冊時的訊息提示功能

過程中碰到什麼困難？

在這個專案我花最多時間的部分是將資料的處理，包括 Date Object 格式的轉換和 mongoose populate 的使用，閱讀官方文件大致上都能解決問題。
